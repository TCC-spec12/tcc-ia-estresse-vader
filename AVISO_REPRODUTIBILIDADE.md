# Nota de reprodutibilidade

Este repositório contém o código compatível com a metodologia descrita no
trabalho original. A base integral de aproximadamente 6.000 comentários não
estava anexada ao documento recebido.

Por esse motivo:

1. o código pode ser apresentado como implementação do procedimento;
2. os seis trechos incluídos servem apenas para teste;
3. os resultados exatos do trabalho dependem da amostra original;
4. não se deve afirmar que o teste de seis frases reproduz os resultados da
   amostra completa;
5. ao executar com arquivos Pushshift, o total efetivamente obtido deve ser
   informado sem completar registros artificialmente.
